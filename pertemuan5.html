ðŸ“˜ KONSEP FORM & HTTP METHOD

Form HTML digunakan untuk mengirim data ke server.

  • Form memiliki atribut action (tujuan pengiriman data) dan method (metode pengiriman: GET atau POST).
  • GET: Data dikirim melalui URL, terlihat di address bar. Cocok untuk data non-sensitif.
  • POST: Data dikirim melalui body HTTP, tidak terlihat di URL. Cocok untuk data sensitif.
Atribut Form Deskripsi
action Menentukan URL atau file PHP yang akan memproses data form.
method Menentukan metode pengiriman data (GET atau POST).
Perbedaan Metode GET dan POST
Method Keterangan
GET Data tampil di URL
POST Data tidak tampil di URL
ðŸ“˜ FORM GET

Contoh form menggunakan metode GET:

Buat file: form_get.php

<form action="process.php" method="get">
  <label for="name">Name:</label>
  <input type="text" id="name" name="name">
  <input type="submit" value="Submit">
</form>

Buat file: process.php
di file process.php, kita dapat mengambil data menggunakan $_GET:

<?php
if ($_SERVER["REQUEST_METHOD"] == "GET") {
    $name = $_GET["name"];
    echo "Hello, " . htmlspecialchars($name);
}

Akses: http://localhost/belajarphp/form_get.php,
Isi form, dan klik tombol Submit.
Data akan terlihat di URL seperti: http://localhost/belajarphp/process.php?name=YourName

ðŸ“˜ FORM POST

Contoh form menggunakan metode POST:

Buat file: form_post.php 
<form action="process_post.php" method="post">
  Username: <input type="text" name="username"><br>
  Password: <input type="password" name="password"><br>
  <button type="submit">Login</button>
</form>

Buat file: process_post.php
Di file process_post.php, kita dapat mengambil data menggunakan $_POST:

<?php
if ($_SERVER["REQUEST_METHOD"] == "POST") {
    $username = $_POST["username"];
    $password = $_POST["password"];
    echo "Username: " . htmlspecialchars($username) . "<br>";
    echo "Password: " . htmlspecialchars($password);
}
?>

Akses: http://localhost/belajarphp/form_post.php,
Isi form, dan klik tombol Submit.
Data tidak akan terlihat di URL.

ðŸ“˜ VALIDASI DASAR INPUT PENGGUNA

Validasi dasar dapat dilakukan untuk memastikan data yang diinputkan sesuai dengan yang diharapkan.

Contoh validasi sederhana pada form login:

Buat file: validasi_login.php 
<?php
if ($_SERVER["REQUEST_METHOD"] == "POST") {
    $username = $_POST["username"];
    $password = $_POST["password"];

    if (empty($username) || empty($password)) {
        echo "Username dan password harus diisi.";
    } else {
        echo "Username: " . htmlspecialchars($username) . "<br>";
        echo "Password: " . htmlspecialchars($password);
    }
}
?>

Dengan validasi ini, jika pengguna mengirimkan form tanpa mengisi username atau password, mereka akan menerima pesan kesalahan.

ðŸ“˜ LATIHAN: FORM REGISTRASI SEDERHANA

Buat form registrasi sederhana yang meminta nama, email, dan password. Validasi agar semua field harus diisi sebelum data dikirim.

Buat file: register.php 
<form action="proses_register.php" method="post">
  Nama: <input type="text" name="nama"><br>
  Email: <input type="text" name="email"><br>
  Password: <input type="password" name="password"><br>
  <button type="submit">Daftar</button>
</form>
Buat file: proses_register.php 
<?php
if ($_SERVER["REQUEST_METHOD"] == "POST") {
    $nama = $_POST["nama"];
    $email = $_POST["email"];
    $password = $_POST["password"];

    if (empty($nama) || empty($email) || empty($password)) {
        echo "Semua field harus diisi.";
    } else {
        echo "Registrasi berhasil!<br>";
        echo "Nama: " . htmlspecialchars($nama) . "<br>";
        echo "Email: " . htmlspecialchars($email);
    }
}
?>

Akses: http://localhost/belajarphp/register.php,
Isi form, dan klik tombol Daftar.
Jika ada field yang kosong, akan muncul pesan kesalahan.
