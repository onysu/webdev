ğŸ“˜ Konsep Database

Database adalah kumpulan data yang terorganisir dan dapat diakses dengan mudah. MySQL adalah sistem manajemen basis data relasional (RDBMS) yang populer digunakan bersama PHP untuk menyimpan dan mengelola data aplikasi web.

Tabel adalah struktur dalam database yang menyimpan data dalam baris dan kolom. Setiap tabel memiliki nama dan terdiri dari beberapa kolom yang mendefinisikan jenis data yang disimpan.

Database = tempat menyimpan data secara terstruktur.
Istilah penting:

  • Database â†’ lemari
  • Table â†’ laci
  • Field â†’ kolom
  • Record â†’ baris data

Contoh:

  • Database: db_siswa
  • Table: siswa
  • Field/Kolom: id, nama, kelas, alamat

ğŸ“˜ Membuat Database

Langkah-langkah membuat database di phpMyAdmin:

  1. Buka phpMyAdmin melalui XAMPP.
  2. Klik tab "Databases" atau "Basis data".
  3. Masukkan nama database, misalnya db_siswa.
  4. Pilih collation (misalnya utf8_general_ci) dan klik "Create" atau "Buat".

ğŸ“˜ Membuat Tabel

Langkah-langkah membuat tabel di phpMyAdmin:

  1. Pilih database yang telah dibuat.
  2. Klik tab "Structure" lalu "Create table".
  3. Masukkan nama tabel, misalnya siswa, dan jumlah kolom.
  4. Definisikan kolom seperti id (INT, Primary Key, AI), nama (VARCHAR), kelas (VARCHAR), alamat (TEXT).

    Penjelasan:

    Di phpMyAdmin, PK (Primary Key) adalah kolom unik untuk mengidentifikasi setiap baris data, sedangkan AI (Auto Increment) adalah fitur yang membuat nilai kolom PK otomatis bertambah setiap kali ada data baru, sangat berguna untuk kolom ID agar tidak perlu diisi manual. Keduanya sering digunakan bersamaan pada kolom ID untuk memastikan setiap data memiliki pengenal unik yang otomatis bertambah, misalnya 1, 2, 3, dst..

  5. Klik "Save" atau "Simpan" yang berada di bawah untuk membuat tabel.

ğŸ“˜ Menyimpan Data (CREATE)

Contoh kode untuk menyimpan data dari form ke database:

<?php
if ($_SERVER["REQUEST_METHOD"] == "POST") {
    $nama = $_POST['nama'];
    $kelas = $_POST['kelas'];
    $alamat = $_POST['alamat'];
    $sql = "INSERT INTO siswa (nama, kelas, alamat) VALUES ('$nama', '$kelas', '$alamat')";
    if ($conn->query($sql) === TRUE) {
        echo "New record created successfully";
    } else {
        echo "Error: " . $sql . "<br>" . $conn->error;
    }
}
?>

Penjelasan:

  1. if ($_SERVER["REQUEST_METHOD"] == "POST") {

    Mengecek apakah form telah dikirim dengan metode POST.
    Jika ya, maka kode di dalam { } akan dijalankan.

  2. $nama = $_POST['nama'];

    Mengambil data dari form input dengan nama "nama" dan menyimpannya ke variabel $nama.

  3. $kelas = $_POST['kelas'];

    Mengambil data dari form input dengan nama "kelas" dan menyimpannya ke variabel $kelas.

  4. $alamat = $_POST['alamat'];

    Mengambil data dari form input dengan nama "alamat" dan menyimpannya ke variabel $alamat.

  5. $sql = "INSERT INTO siswa (nama, kelas, alamat) VALUES ('$nama', '$kelas', '$alamat')";

    Membuat query SQL untuk menyisipkan data baru ke tabel siswa.

  6. if ($conn->query($sql) === TRUE) {

    Menjalankan query SQL. Jika berhasil, kode di dalam { } akan dijalankan.

  7. echo "New record created successfully";

    Menampilkan pesan bahwa data baru berhasil disimpan.

  8. } else {

    Jika query gagal, kode di dalam { } akan dijalankan.

  9. echo "Error: " . $sql . "<br>" . $conn->error;

    Menampilkan pesan error beserta detailnya.

  10. }

    Menandai akhir dari blok kode if.

  11. }

    Menandai akhir dari blok kode utama.

  12. ?>

    Ini adalah tanda akhir kode PHP.

Kesimpulan:

  • Kode di atas menyimpan data dari form ke tabel siswa di database.
  • Data diambil dari form menggunakan metode POST dan dimasukkan ke dalam query SQL INSERT.
  • Jika penyimpanan berhasil, pesan sukses ditampilkan; jika gagal, pesan error ditampilkan.

ğŸ“˜ Menampilkan Data (READ)

Contoh kode untuk menampilkan data dari database:

<?php
$sql = "SELECT id, nama, kelas, alamat FROM siswa";
$result = $conn->query($sql);
if ($result->num_rows > 0) {
    echo "<table><tr><th>ID</th><th>Nama</th><th>Kelas</th><th>Alamat</th></tr>";
    while($row = $result->fetch_assoc()) {
        echo "<tr><td>" . $row["id"]. "</td><td>" . $row["nama"]. "</td><td>" . $row["kelas"]. "</td><td>" . $row["alamat"]. "</td></tr>";
    }
    echo "</table>";
} else {
    echo "0 results";
}
?>

Penjelasan:

  1. $sql = "SELECT id, nama, kelas, alamat FROM siswa";

    Membuat query SQL untuk mengambil data dari tabel siswa.

  2. $result = $conn->query($sql);

    Menjalankan query SQL dan menyimpan hasilnya ke variabel $result.

  3. if ($result->num_rows > 0) {

    Mengecek apakah ada data yang diambil. Jika ada, kode di dalam { } akan dijalankan.

  4. echo "<table><tr><th>ID</th><th>Nama</th><th>Kelas</th><th>Alamat</th></tr>";

    Menampilkan header tabel HTML.

  5. while($row = $result->fetch_assoc()) {

    Looping melalui setiap baris data yang diambil.

  6. echo "<tr><td>" . $row["id"]. "</td><td>" . $row["nama"]. "</td><td>" . $row["kelas"]. "</td><td>" . $row["alamat"]. "</td></tr>";

    Menampilkan setiap baris data dalam format tabel HTML.

  7. }

    Menandai akhir dari blok kode while.

  8. echo "</table>";

    Menutup tag tabel HTML.

  9. } else {

    Jika tidak ada data, kode di dalam { } akan dijalankan.

  10. echo "0 results";

    Menampilkan pesan bahwa tidak ada data.

  11. }

    Menandai akhir dari blok kode if.

  12. ?>

    Ini adalah tanda akhir kode PHP.
