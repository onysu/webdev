IF Sederhana

Struktur dasar dari percabangan IF adalah sebagai berikut:

<?php
if (kondisi) {
    // kode yang dieksekusi jika kondisi benar
}
?>

Contoh penggunaan IF sederhana:

<?php
$nilai = 85;

if ($nilai >= 80) {
    echo "Nilai Anda A";
}
?>

Dalam contoh di atas, jika nilai lebih besar atau sama dengan 80, maka akan menampilkan "Nilai Anda A".

IF â€“ ELSE

Struktur dasar dari percabangan IF â€“ ELSE adalah sebagai berikut:

<?php
if (kondisi) {
    // kode yang dieksekusi jika kondisi benar
} else {
    // kode yang dieksekusi jika kondisi salah
}
?>

Contoh penggunaan IF â€“ ELSE:

<?php
$nilai = 75;

if ($nilai >= 60) {
    echo "Lulus";
} else {
    echo "Tidak Lulus";
}
?>

Dalam contoh di atas, jika nilai lebih besar atau sama dengan 60, maka akan menampilkan "Lulus". Jika tidak, akan menampilkan "Tidak Lulus".

ELSEIF

Struktur dasar dari percabangan ELSEIF adalah sebagai berikut:

<?php
if (kondisi1) {
    // kode yang dieksekusi jika kondisi1 benar
} elseif (kondisi2) {
    // kode yang dieksekusi jika kondisi2 benar
} else {
    // kode yang dieksekusi jika semua kondisi salah
}
?>

Contoh penggunaan ELSEIF:

<?php
$nilai = 85;

if ($nilai >= 80) {
    echo "Nilai Anda A";
} elseif ($nilai >= 70) {
    echo "Nilai Anda B";
} else {
    echo "Nilai Anda C";
}
?>

Dalam contoh di atas, jika nilai lebih besar atau sama dengan 80, maka akan menampilkan "Nilai Anda A". Jika nilai antara 70 dan 79, maka akan menampilkan "Nilai Anda B". Jika nilai kurang dari 70, maka akan menampilkan "Nilai Anda C".

ðŸ“˜ E. PERCABANGAN SWITCH

Struktur dasar dari percabangan SWITCH adalah sebagai berikut:

<?php
switch (kondisi) {
    case nilai1:
        // kode yang dieksekusi jika kondisi sama dengan nilai1
        break;
    case nilai2:
        // kode yang dieksekusi jika kondisi sama dengan nilai2
        break;
    default:
        // kode yang dieksekusi jika tidak ada case yang cocok
}
?>

Contoh penggunaan SWITCH:

<?php
$hari = "Senin";

switch ($hari) {
    case "Senin":
        echo "Hari pertama dalam seminggu";
        break;
    case "Selasa":
        echo "Hari kedua dalam seminggu";
        break;
    default:
        echo "Hari lainnya";
}
?>
Catatan:
  • Setiap case diakhiri dengan pernyataan break; untuk menghentikan eksekusi setelah case yang cocok ditemukan.
  • Bagian default: bersifat opsional dan akan dieksekusi jika tidak ada case yang cocok.

Dalam contoh di atas, jika hari adalah "Senin", maka akan menampilkan "Hari pertama dalam seminggu". Jika hari adalah "Selasa", maka akan menampilkan "Hari kedua dalam seminggu". Jika hari bukan "Senin" atau "Selasa", maka akan menampilkan "Hari lainnya".

ðŸ“˜ PERULANGAN (LOOP) PHP

Perulangan memungkinkan kita untuk mengeksekusi blok kode berulang kali selama kondisi tertentu terpenuhi. Berikut adalah beberapa jenis perulangan yang umum digunakan dalam PHP:

For

Digunakan ketika jumlah iterasi sudah diketahui sebelumnya. 
<?php
for ($i = 1; $i <= 5; $i++) {
    echo "Perulangan ke-$i <br>";
}
?>

While

Digunakan ketika jumlah iterasi tidak diketahui dan bergantung pada kondisi tertentu. 
<?php
$i = 1;

while ($i <= 5) {
    echo "Perulangan ke-$i <br>";
    $i++;
}
?>

Do-While

Mirip dengan while, tetapi kode akan dieksekusi setidaknya sekali sebelum kondisi diperiksa. 
<?php
$i = 1;

do {
    echo "Perulangan ke-$i <br>";
    $i++;
} while ($i <= 5);
?>

FOREACH (Array)

Digunakan khusus untuk mengiterasi elemen dalam array. 
<?php
$buah = array("Apel", "Pisang", "Jeruk");
foreach ($buah as $item) {
    echo "Buah: $item <br>";
}
?>

Dengan memahami dan menguasai percabangan serta perulangan dalam PHP, Anda dapat membuat program yang lebih dinamis dan responsif terhadap berbagai kondisi. Latihanlah dengan membuat beberapa contoh kode untuk memperkuat pemahaman Anda!
